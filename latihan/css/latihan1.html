Praktik CSS - Background Properties

1. Background Repeat

Property background-repeat mengontrol bagaimana gambar latar belakang diulang pada elemen.

Property Value Output
background-repeat repeat
background-repeat repeat-x
background-repeat repeat-y
background-repeat no-repeat

2. Background Position

Property background-position mengontrol posisi gambar latar belakang dalam elemen.

Position Output
top left
top center
top right
center left
center center
center right
bottom left
bottom center
bottom right
