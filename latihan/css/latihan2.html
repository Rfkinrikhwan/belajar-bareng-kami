Dokumentasi CSS - Text Styling

Referensi lengkap untuk berbagai properti CSS yang digunakan untuk styling text pada elemen HTML. Setiap contoh disertai dengan visual untuk mempermudah pemahaman.

Color Properties

Properti color digunakan untuk mengatur warna teks pada elemen.

Property Value Output
color red
Contoh teks merah
green
Contoh teks hijau
blue
Contoh teks biru
darkslategray
Contoh teks abu gelap

Letter Spacing

Properti letter-spacing mengatur jarak antar huruf dalam teks.

Property Value Output
letter-spacing normal
Teks tanpa jarak tambahan
5px
T e k s d e n g a n j a r a k

Text Align

Properti text-align mengatur perataan horizontal teks dalam elemen.

Property Value Output
text-align left
Teks rata kiri
center
Teks rata tengah
right
Teks rata kanan
justify
Teks ini disusun agar memenuhi lebar kontainer, menggunakan penyusunan justify agar teksnya terlihat rapi di kedua sisi. Tambahan kata untuk menunjukkan efek justify yang lebih jelas.

Text Decoration

Properti text-decoration menambahkan dekorasi pada teks seperti garis bawah, atas, atau coret.

Property Value Output
text-decoration none
Tanpa dekorasi
underline
Teks bergaris bawah
overline
Teks bergaris atas
line-through
Teks dicoret
blink
Teks blink (tidak didukung sebagian besar browser)

Text Indent

Properti text-indent mengatur indentasi baris pertama dalam blok teks.

Property Value Output
text-indent 40px
Paragraf ini memiliki indentasi awal 40px. Ini dapat membantu membuat paragraf terlihat lebih terstruktur.
10%
Paragraf ini menggunakan indentasi 10% dari lebar elemen. Indentasi persentase akan menyesuaikan dengan lebar elemen.

Text Transform

Properti text-transform mengontrol kapitalisasi teks.

Property Value Output
text-transform capitalize
setiap awal kata besar seperti ini
uppercase
teks huruf besar semua
lowercase
TEKS DIKECILKAN SEMUA
none
Teks apa adanya

Property Value Output

Teks dengan Arial
"Comic Sans MS", cursive
Teks dengan Comic Sans
Georgia, serif
Teks dengan Georgia

Property Value Output

24px
Teks 24px

Teks 150%

Font Style

Properti font-style mengatur gaya miring dari teks.

Property Value Output
font-style normal
Teks normal
italic
Teks miring italic
oblique
Teks miring oblique

Font Weight

Properti font-weight mengatur ketebalan dari font.

Property Value Output
